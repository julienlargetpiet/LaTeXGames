# LaTeXGames

## DooMTeX

### Usage

```
$ bash loop.sh
```

And in another terminal view main.pdf with your favorite pdf viewer.

```
$ zathura main.pdf
```

- "j" = move backward
- "k" = move forward
- "h" = turn to the left opening
- "l" = turn to the right opening
- "r" = reset
- "s" = shoot
- "q" = quit
